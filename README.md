# TweetChart v0.0.1.1

  * `bundle` to install required gems
  * `bundle exec guard` to watch coffeescript
  * `npm install` to install required modules
  * `./bin/devserver` to start the server

## Run server

  `./bin/devserver`

## Run in mock mode

  `./bin/devserver app -m`

## Run tests

  `./bin/test` or auto test `./bin/test -auto`
